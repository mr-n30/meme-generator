# Meme Generator
A Meme Generator project created from a FreeCodeCamp YouTube video to learn React

---

## Install
```
git clone https://github.com/mr-n30/meme-generator.git
cd meme-generator
npm install
```

---

## Usage
```
npm run dev
```
